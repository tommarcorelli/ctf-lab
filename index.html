🧠 Buffer overflow (pédagogique)

Une fonction lit une entrée dans char buf[16] sans vérifier la taille. La pile contient ensuite le RBP sauvé (8 o) puis l'adresse de retour (8 o, offset 24). Fais déborder le buffer pour écraser l'adresse de retour avec l'adresse de win() = 0x401156. Aucun code n'est réellement exécuté — c'est un schéma pédagogique.
